# Desafio Nginx e Node JS -> fc3.0
Desafio de docker : nginx e node.js

Curso FC 3.0
(deixe os dois arquivos na mesma pasta)

# Instruções:

1.  Faça o clone deste repositório.
2.  Execute o terminal ( na pasta raiz do projeto )
3.  digite o seguinte comando:
   ```
    docker-compose up -d
   ```
( O docker irá buildar, através do arquivo docker-compose.yaml )

4. Abra algum navegador de internet e digite na barra de endereços:

   ```
   http://localhost:8080
   ```

5. Se tudo der certo, você deve ver o seguinte resultado, ex:

```
Full Cycle Rocks!
Pessoa-72
```
